# Scrum Guide Anatomy

**スクラムガイドの解剖図 — すべてのルールを「なぜ？」で三本柱につなぐ**

スクラムガイド 2020 の各記述を分類・注釈し、経験主義の三本柱（透明性・検査・適応）との接続を明示するリファレンスビューアーです。

https://pryo-46.github.io/scrum-guide-anatomy/

## 機能

- **タグ別フィルタ** — 公理・定義・ルール・根拠・許容・実践例・慣行で絞り込み
- **三本柱フィルタ** — T（透明性）・I（検査）・A（適応）で絞り込み
- **キーワード検索** — テキスト内容からヒット箇所をハイライト
- **目次ナビゲーション** — 章・節へのジャンプ
- **「なぜ？」の展開** — 各ルールをクリックすると三本柱との接続理由を表示

## ローカルで動かす

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`dist/` にビルド成果物が出力されます。

## フィードバック

タグの分類（ルール / 定義 / 根拠 など）や三本柱への接続は、スクラムガイドの解釈に基づく著者の判断です。

「この分類は違うのでは」「この柱への接続はこう考えるべきでは」といったご指摘は歓迎します。
Issue からお寄せください。
